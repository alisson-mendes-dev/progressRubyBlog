# 💎 ProgressBlog

Bem-vindo ao **ProgressBlog**, um espaço dedicado a documentar minha jornada de aprendizado na linguagem Ruby. Este projeto foi desenvolvido para ser um diário de bordo técnico, onde registro desafios, conceitos aprendidos e evoluções no ecossistema Ruby/Sinatra.

---

## 🚀 O Projeto

O **ProgressBlog** nasceu da necessidade de organizar o conhecimento. Com uma interface limpa, focada no tema escuro ("Dark Mode"), o blog permite que eu gerencie postagens de forma rápida através de um painel administrativo exclusivo, garantindo que o foco total esteja no aprendizado.

### ✨ Principais Funcionalidades

* **Tema Dark & Ruby Inspired**: Design sofisticado com paleta de cores inspirada na linguagem Ruby.
* **Gerenciamento de Posts**: CRUD completo (Criar, Ler, Atualizar, Deletar) via painel administrativo.
* **Busca em Tempo Real (AJAX)**: Encontre seus registros rapidamente com uma busca dinâmica.
* **Paginação Otimizada**: Navegação eficiente, carregando 6 posts por página para manter a performance.
* **Segurança**: Área administrativa protegida com autenticação.
* **Metadados**: Registro automático de data e autoria em cada postagem.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando o ecossistema Ruby:

* **Linguagem**: Ruby
* **Framework Web**: [Sinatra](http://sinatrarb.com/)
* **Banco de Dados**: MongoDB (via [Mongoid](https://docs.mongodb.com/mongoid/))
* **Frontend**: HTML5, CSS3 (com Flexbox/Grid) e JavaScript (AJAX/Fetch API)
* **Ícones**: [Font Awesome](https://fontawesome.com/)
